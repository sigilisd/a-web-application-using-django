# Цель
Создать веб-приложение для публикации новостей с использованием Django, SQLite в качестве базы данных и uv в качестве менеджера пакетов.

# Технические требования
# 1. Установка и настройка
* Использовать uv в качестве менеджера пакетов и виртуального окружения

* Использовать SQLite в качестве базы данных

* Настроить проект с использованием переменных окружения для конфиденциальных данных

* Создать подробный README.md с инструкциями по запуску

# 2. Модели данных
Создать следующие модели в приложении news:

# Новость (Article)
* title (CharField) - заголовок новости

* content (TextField) - содержимое новости

* author (ForeignKey) - связь с User (автор)

* created_at (DateTimeField) - дата создания (автозаполнение)

* updated_at (DateTimeField) - дата обновления (автообновление)

* is_published (BooleanField) - статус публикации

# 3. Функциональные требования
# Административная часть
1. Панель администратора Django с возможностью:

  * Управления новостями

# Публичная часть
1. Главная страница:

  * Список последних новостей

  * Фильтрация по заголовку

2. Страница новости:

  * Полное содержание новости

  * Информация об авторе и дате публикации


# 4. Дополнительные требования
* Использовать классы представления (Class-Based Views)

* Добавить простой CSS для базового оформления внутрь шаблона base.html
